Page Information

Start Placing Markers: This button allows you to place markers on the map to mark birdwatching spots. You can click anywhere on the map to add a marker.

Locate Me: This button uses your current location and centers the map on it, making it easier for you to find your position.

Start Drawing Rectangles: This button lets you draw rectangular areas on the map. These can be used to see location specific data for birdwatching.

Clear All: This button clears all the markers and rectangles you have placed on the map. Use it if you want to reset the map.

Go to Stats: This button redirects you to the statistics page where you can view your birdwatching data.

Go to Checklists: This button takes you to the checklists page where you can add a checklist.

Filter by Species: This search bar allows you to filter species and select the ones you're interested in. When Enter is pressed, it will update the density map.

Selected Species: This section shows a list of species you've selected using checkboxes. You can deselect them here and start a new selection.
